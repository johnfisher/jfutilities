# What's this ZBIN Project?


This is a place to store scripts and utilities which may be useful outisde rdr. 
Its suggested that you check this project out in some sensible place and put
zbin in your path. 
Have cron update it every day.

Note: its unlikely that any one script will work on both Windows and Linux,
and/or Solaris et. al.

# Brief descriptions of some of the scripts:

zdmp.sh		Makes formatted dump of kernel virtual memory.  Uses zreg.
